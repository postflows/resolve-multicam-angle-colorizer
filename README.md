# Multicam Angle Colorizer

> Part of [PostFlows](https://github.com/postflows) toolkit for DaVinci Resolve

Colorize multicam timeline clips by angle using Resolve clip colors.

## What it does

Detects angles from clip names (Angle 1, Cam2, Video 1, etc.), maps them to Resolve clip colors. Modes: Automatic (default palette), Manual (per-angle color ComboBoxes), Individual (single angle + color). Works on video and audio tracks.

## Requirements

- DaVinci Resolve 18+
- Open project and timeline

## Installation

Copy the script to:

- **macOS:** `~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/`
- **Windows:** `C:\ProgramData\Blackmagic Design\DaVinci Resolve\Fusion\Scripts\`

## Usage

Open a multicam timeline, run script. Select mode, set colors if Manual/Individual, click Colorize.
